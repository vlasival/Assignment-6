# Assignment-6

## Информация о препарате 

### 1. Действующее вещество

Левотироксин (Levothyroxine, L-thyroxine, T4) — синтетический аналог гормона щитовидной железы тироксина. В лекарственных формах чаще используется левотироксин натрия.

### 2. Область применения

Левотироксин применяется как препарат заместительной терапии при состояниях, связанных с недостаточной функцией щитовидной железы (гипотиреоз). В клинической практике также может использоваться для TSH-супрессивной терапии по назначению врача в отдельных ситуациях (например, после лечения некоторых заболеваний щитовидной железы).

### 3. Молекулярный механизм действия

Левотироксин (T4) является преимущественно прогормоном: в тканях организма он частично превращается в трийодтиронин (T3). T3 связывается с ядерными тиреоидными рецепторами (TRα/TRβ) и регулирует экспрессию множества генов, влияя на обмен веществ, рост и развитие тканей. Это относят к геномному (транскрипционному) механизму действия тиреоидных гормонов.
